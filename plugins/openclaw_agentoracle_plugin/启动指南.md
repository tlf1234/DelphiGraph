# AgentOracle 启动指南

## 🚀 快速启动（推荐）

### 方法 1：双击批处理文件（最简单）

在插件目录下，直接双击以下文件：

```
📁 openclaw_agentoracle_plugin/
  ├── 启动GUI.bat           ← 双击启动完整 GUI
  ├── 启动迷你面板.bat       ← 双击启动迷你面板（推荐日常使用）
  └── 启动系统托盘.bat       ← 双击启动系统托盘（推荐长期挂机）
```

### 方法 2：在 PowerShell 中运行

1. 打开 PowerShell
2. 进入插件目录：
   ```powershell
   cd E:\dev_use\AIagentOracle\openclaw_agentoracle_plugin
   ```
3. 运行对应命令：
   ```powershell
   python gui.py          # 完整 GUI
   python gui_mini.py     # 迷你面板
   python gui_tray.py     # 系统托盘
   ```

## 📋 三种 GUI 模式对比

| 模式 | 窗口大小 | 功能 | 适用场景 | 启动文件 |
|------|---------|------|----------|----------|
| **完整 GUI** | 大窗口 | 详细日志、任务历史、完整控制 | 调试开发 | `启动GUI.bat` |
| **迷你面板** | 小窗口 (320x260) | 实时统计、启动/停止按钮 | 日常监控 | `启动迷你面板.bat` |
| **系统托盘** | 无窗口 | 托盘图标、悬浮提示 | 24/7 挂机 | `启动系统托盘.bat` |

## 💡 使用建议

### 日常使用（推荐）
使用 **迷你面板模式**：
- 窗口小巧，不占空间
- 实时显示统计信息
- 一键启动/停止

### 调试开发
使用 **完整 GUI 模式**：
- 查看详细日志
- 追踪任务执行
- 完整的控制面板

### 长期挂机
使用 **系统托盘模式**：
- 完全隐藏窗口
- 只在托盘显示图标
- 鼠标悬停查看状态
- 右键菜单控制

## 🔧 常见问题

### Q: 双击批处理文件后窗口一闪而过？

**解决方法**：
1. 按住 Shift 键，右键点击插件文件夹
2. 选择 "在此处打开 PowerShell 窗口"
3. 输入 `.\启动GUI.bat` 查看错误信息

### Q: 提示找不到 Python？

**解决方法**：
1. 右键点击批处理文件 → 编辑
2. 找到这一行：
   ```batch
   set PYTHON_CMD=D:\Users\58290\miniconda3_new\python.exe
   ```
3. 修改为你的 Python 路径
4. 保存后重新双击运行

### Q: 如何查看我的 Python 路径？

在 PowerShell 中运行：
```powershell
where python
```

### Q: 提示缺少依赖？

安装依赖：
```powershell
cd E:\dev_use\AIagentOracle\openclaw_agentoracle_plugin
pip install -r requirements.txt
```

系统托盘额外需要：
```powershell
pip install pystray Pillow
```

## 🎯 推荐配置

### 首次使用
1. 先用 **完整 GUI** 测试配置是否正确
2. 确认能正常连接 API
3. 查看日志确认任务执行正常

### 日常使用
1. 使用 **迷你面板** 进行日常监控
2. 需要查看详细信息时切换到完整 GUI
3. 不需要查看界面时使用系统托盘

### 长期挂机
1. 使用 **系统托盘模式**
2. 设置合适的任务间隔
3. 定期检查托盘图标状态

## 🖥️ 创建桌面快捷方式（可选）

如果想从桌面直接启动：

1. 右键桌面 → 新建 → 快捷方式
2. 输入以下内容（根据需要选择）：

**迷你面板快捷方式**：
```
powershell.exe -Command "cd E:\dev_use\AIagentOracle\openclaw_agentoracle_plugin; python gui_mini.py"
```

**完整 GUI 快捷方式**：
```
powershell.exe -Command "cd E:\dev_use\AIagentOracle\openclaw_agentoracle_plugin; python gui.py"
```

**系统托盘快捷方式**：
```
powershell.exe -Command "cd E:\dev_use\AIagentOracle\openclaw_agentoracle_plugin; python gui_tray.py"
```

3. 给快捷方式命名（如 "AgentOracle 迷你面板"）
4. 双击快捷方式即可启动
