# AgentOracle 绿色版打包指南

## 什么是绿色版？

绿色版是一个完全便携的软件包，用户无需安装 Python 或任何依赖，解压即可使用。

## 核心原理

使用 **Python Embeddable Package**（Windows）或 **虚拟环境**（Mac/Linux），将 Python 运行时和所有依赖打包到一个文件夹中。

## 优势

✓ **零门槛**: 用户不需要懂 Python，下载解压即用  
✓ **绝对安全**: 不是编译的黑盒 .exe，是官方 python.exe + 明文 .py 脚本  
✓ **不污染系统**: 所有文件在一个文件夹，删除即卸载  
✓ **完全开源**: 所有源代码可见可修改  
✓ **杀毒友好**: 不会被杀毒软件误报

## 打包步骤

### Windows 版本

1. 运行打包脚本:
   ```bash
   打包绿色版.bat
   ```

2. 按照提示下载 Python Embeddable Package:
   - 访问: https://www.python.org/downloads/windows/
   - 下载对应版本的 "Windows embeddable package (64-bit)"
   - 放到当前目录

3. 脚本会自动:
   - 解压 Python 环境
   - 配置 site-packages 支持
   - 安装所有依赖
   - 复制插件文件
   - 创建启动脚本

4. 完成后得到 `AgentOracle_Green` 文件夹

### Mac/Linux 版本

1. 运行打包脚本:
   ```bash
   chmod +x 打包绿色版_Mac.sh
   ./打包绿色版_Mac.sh
   ```

2. 脚本会自动完成所有步骤

3. 完成后得到 `AgentOracle_Green` 文件夹

## 目录结构

```
AgentOracle_Green/
  ├── python_env/              # Python 环境
  │   ├── python.exe           # (Windows)
  │   ├── pythonw.exe          # (Windows 无窗口)
  │   └── Lib/site-packages/  # 所有依赖
  │
  ├── plugin/                  # 插件源代码
  │   ├── gui.py
  │   ├── config.json
  │   └── docs/
  │
  ├── 启动AgentOracle.bat      # Windows 启动
  ├── 启动AgentOracle.command  # Mac 启动
  └── 使用说明.txt
```

## 分发方式

1. 压缩文件夹:
   - Windows: `AgentOracle_Green_Windows.zip`
   - Mac: `AgentOracle_Green_Mac.tar.gz`
   - Linux: `AgentOracle_Green_Linux.tar.gz`

2. 上传到 GitHub Release

3. 用户下载解压即可使用

## 用户使用流程

1. 下载对应平台的压缩包
2. 解压到任意位置
3. 编辑 `plugin/config.json` 配置 API 信息
4. 双击启动脚本（Windows）或运行 .sh/.command（Mac/Linux）
5. 完成！

## 注意事项

### 平台差异

**Windows 版本**:
- ✓ 使用 Embeddable Package，完全独立
- ✓ 用户无需安装 Python
- ✓ 真正的"绿色版"

**Mac 版本**:
- ⚠️ 使用虚拟环境，依赖系统 Python
- ✓ macOS 10.15+ 自带 Python 3，大部分用户可直接使用
- ⚠️ 技术上不是完全独立，但实际使用无障碍

**Linux 版本**:
- ⚠️ 使用虚拟环境，依赖系统 Python
- ⚠️ 用户需要确保系统已安装 Python 3
- ⚠️ 大部分发行版自带 Python，但不保证

### 建议

- Windows 用户：推荐使用绿色版（完全独立）
- Mac 用户：可以使用绿色版（依赖系统 Python，但通常已有）
- Linux 用户：建议提供安装脚本或 Docker 镜像
- 首次打包后建议在目标平台测试所有功能

## 技术细节

### Windows Embeddable Package 配置

修改 `python3xx._pth` 文件启用 site-packages:
```
python3xx.zip
.
../plugin

# 启用 site-packages
import site
```

### 启动脚本原理

Windows:
```batch
python_env\pythonw.exe plugin\gui.py
```

Mac/Linux:
```bash
source python_env/bin/activate
python plugin/gui.py
```

## 常见问题

**Q: 为什么不用 PyInstaller？**  
A: PyInstaller 编译成黑盒 .exe，容易被杀毒软件误报，且不开源透明。

**Q: 绿色版体积多大？**  
A: Windows 约 50-80MB，Mac/Linux 约 30-50MB。

**Q: 可以跨平台使用吗？**  
A: 不可以，需要为每个平台单独打包。

**Q: 用户可以修改源代码吗？**  
A: 可以！所有源代码都在 plugin/ 目录，完全开源。
