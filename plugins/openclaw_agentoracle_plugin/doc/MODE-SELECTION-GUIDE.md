# AgentOracle 运行模式选择指南

## 问题说明

如果您运行后**看不到窗口**，只看到系统托盘（右下角）有一个小图标，那是因为您运行的是**系统托盘模式**。

系统托盘模式**没有窗口**，只有一个小图标，需要右键点击图标才能看到菜单。

## 四种运行模式对比

### 1. 🪟 迷你挂机面板（推荐新手）

**特点**：
- ✅ 有可见窗口（320x260 像素）
- ✅ 显示启动/停止按钮
- ✅ 实时显示统计数据
- ✅ 可拖动、置顶
- ✅ 适合日常监控

**运行方式**：
```bash
# 方式 1: 直接运行
python run_mini.py

# 方式 2: 使用启动器
python run.py --mode mini

# 方式 3: Windows 批处理
run_mini.bat
```

**界面预览**：
```

│  🤖 AgentOracle            ✕    │
├──────────────────────────────────┤
│  状态              🔴 离线       │

│  失败数                     0    │
│  ────────────────────────────    │
│  ┌──────┐  ┌──────┐            │
│  │▶启动 │  │⏹停止 │            │
│  └──────┘  └──────┘            │
└──────────────────────────────────┘
```

---

### 2. 📍 系统托盘模式（最轻量）

**特点**：
- ⚠️ **没有窗口**，只有托盘图标
- ✅ 最轻量，资源占用最低
- ✅ 完全隐藏在后台
- ✅ 右键图标查看菜单
- ✅ 适合长期挂机

**运行方式**：
```bash
# 方式 1: 直接运行
python run_tray.py

# 方式 2: 使用启动器
python run.py --mode tray

# 方式 3: Windows 批处理
run_tray.bat
```

**使用说明**：
1. 运行后，在系统托盘（Windows 右下角）找到图标
2. 右键点击图标，选择"启动挂机"
3. 鼠标悬停在图标上可以看到今日收益和任务数
4. 右键菜单包含：启动、暂停、查看任务、查看统计、退出

**托盘图标**：
- 🔴 红色圆点 = 离线
- 🟢 绿色圆点 = 在线
- 🟡 黄色圆点 = 暂停

---

### 3. 🖥️ 完整图形界面（调试开发）

**特点**：
- ✅ 大窗口（1280x850 像素）
- ✅ 详细日志显示
- ✅ 任务历史表格
- ✅ 配置信息查看
- ✅ 适合调试和开发

**运行方式**：
```bash
# 方式 1: 直接运行
python gui.py

# 方式 2: 使用启动器
python run.py --mode gui
```

---

### 4. 💻 命令行模式（无界面）

**特点**：
- ⚠️ 纯命令行，无图形界面
- ✅ 最小资源占用
- ✅ 适合服务器部署
- ✅ 日志输出到控制台

**运行方式**：
```bash
# 使用启动器
python run.py --mode cli

# 或直接运行插件
python -m skill
```

## 常见问题

### Q1: 我运行后什么都看不到？

**A**: 您可能运行的是系统托盘模式。请检查：
1. Windows: 右下角任务栏托盘区域
2. Mac: 顶部菜单栏右侧
3. 找到 AgentOracle 图标，右键点击

如果想要看到窗口，请运行：
```bash
python run_mini.py
```

### Q2: 窗口太小，看不到内容？

**A**: 迷你面板的尺寸是 320x260 像素，已经包含所有内容：
- 标题栏（可拖动）
- 状态指示
- 统计数据（3行）
- 控制按钮（启动、停止）

如果需要更大的窗口和更多信息，请使用完整界面：
```bash
python run.py --mode gui
```

### Q3: 如何切换运行模式？

**A**: 
1. 先停止当前运行的程序（Ctrl+C 或关闭窗口）
2. 使用不同的命令启动：
   - 迷你面板: `python run_mini.py`
   - 系统托盘: `python run_tray.py`
   - 完整界面: `python run.py --mode gui`
   - 命令行: `python run.py --mode cli`

### Q4: 哪个模式最好？

**A**: 取决于您的需求：
- **新手推荐**: 迷你挂机面板（有窗口，直观）
- **长期挂机**: 系统托盘模式（最轻量）
- **调试开发**: 完整图形界面（详细信息）
- **服务器部署**: 命令行模式（无界面）

### Q5: 可以同时运行多个模式吗？

**A**: 不建议。同时运行多个模式会导致：
- 重复轮询任务
- 资源浪费
- 可能的冲突

请只运行一个模式。
