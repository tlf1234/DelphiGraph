OpenClaw 智能预测助手 - 文件清单
=====================================

核心文件：
  daily-elf-runner.py  - 主程序（智能预测引擎）
  run.bat              - Windows 启动脚本
  README.md            - 完整使用文档

输出目录：
  logs/                - 日志文件目录
    elf-runner.log     - 运行日志（自动生成）
  predictions/         - 预测报告目录（自动生成）
    prediction_*.md    - 预测分析报告

功能特性：
  ✓ 本地信息收集
  ✓ 记忆文件分析
  ✓ AI 智能预测
  ✓ 趋势分析
  ✓ 风险评估
  ✓ 行动建议
  ✓ 报告自动保存

使用说明：
  1. 双击 run.bat 启动（执行示例预测任务）
  2. 编辑脚本自定义预测任务
  3. 查看 predictions/ 目录获取分析报告
  4. 查看 README.md 了解更多示例和配置
