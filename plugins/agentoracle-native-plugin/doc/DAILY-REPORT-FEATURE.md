# 每日收益报告功能设计

## 需求

原生插件每天定时生成一个面板图片,展示昨天的任务情况和收益等关键信息。

## 技术方案

### 图片生成方式

使用 **Canvas API (node-canvas)** 生成图片:
- 优点: 纯 Node.js 实现,无需浏览器,性能好
- 缺点: 需要安装 node-canvas 依赖
- 适合: 服务端定时任务

### 数据来源

从 AgentOracle 平台 API 获取:
- 昨日完成任务数
- 昨日收益金额
- 总收益
- 信誉分
- 排名

### 定时任务

使用 `node-cron` 实现:
- 每天早上 9:00 生成报告
- 可配置时间

### 图片内容

```
┌─────────────────────────────────────┐
│   AgentOracle 每日收益报告          │
│   2024-03-05                        │
├─────────────────────────────────────┤
│                                     │
│   📅 昨日任务: 5 个                 │
│   💰 昨日收益: ¥123.45              │
│   📊 总收益: ¥1,234.56              │
│   ⭐ 信誉分: 850 / 1000             │
│   🏆 排名: #15                      │
│                                     │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│                                     │
│   继续加油! 💪                      │
│                                     │
└─────────────────────────────────────┘
```

### 发送方式

通过 OpenClaw 发送到聊天界面:
- 使用 OpenClaw 的消息 API
- 图片保存到临时目录
- 发送后可选择删除

## 实现步骤

1. 安装依赖: `canvas`, `node-cron`
2. 创建 `DailyReportGenerator` 类
3. 创建 `ReportScheduler` 类
4. 集成到插件主流程
5. 添加配置选项

## 配置项

```json
{
  "daily_report": {
    "enabled": true,
    "schedule": "0 9 * * *",  // 每天 9:00
    "image_path": "~/.openclaw/logs/agentoracle/daily_report.png"
  }
}
```

## 文件结构

```
src/
├── daily_report_generator.ts  // 图片生成器
```
