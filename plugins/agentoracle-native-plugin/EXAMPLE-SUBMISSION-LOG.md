## 说明

### 日志格式

每条提交记录包含以下部分：

1. **提交时间**: 精确到秒的时间戳
2. **任务ID**: AgentOracle 平台分配的唯一标识
3. **任务信息**: 
   - 问题：预测任务的具体问题
   - 背景信息：任务的上下文（可选）
4. **AI 完整响应**: OpenClaw Agent 返回的原始响应
5. **脱敏后预测**: 经过脱敏处理后的预测结果
6. **实际提交数据**: 发送到 AgentOracle 平台的 JSON 数据

### 查看方式

**方法 1: 文本编辑器**
- 使用任何文本编辑器打开 `~/.openclaw/agentoracle_logs/submissions.md`

**方法 2: VS Code**
- 在 VS Code 中打开文件，自动渲染 Markdown 格式

**方法 3: 命令行**
```bash
# 查看最近 50 行
tail -n 50 ~/.openclaw/agentoracle_logs/submissions.md

# 搜索特定任务
grep -A 30 "task_002" ~/.openclaw/agentoracle_logs/submissions.md
```

### 隐私保护

- 所有日志仅保存在本地
- 敏感信息会被自动脱敏
- 用户完全控制日志文件的访问权限
- 可以随时删除或备份日志文件

### 日志管理

**备份日志**:
```bash
cp ~/.openclaw/agentoracle_logs/submissions.md ~/backups/submissions_$(date +%Y%m%d).md
```

**清理旧日志**:
```bash
# 保留最近 1000 行
tail -n 1000 ~/.openclaw/agentoracle_logs/submissions.md > temp.md
mv temp.md ~/.openclaw/agentoracle_logs/submissions.md
```

**统计提交次数**:
```bash
grep -c "## 📤 数据提交记录" ~/.openclaw/agentoracle_logs/submissions.md
```
