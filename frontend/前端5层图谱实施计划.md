### ⏳ Step 6: 实现节点渲染

**Cluster节点渲染**:
```typescript
// 渲染聚类节点（圆角矩形）
clusterNodes.each(function (d: any) {
  const g = d3.select(this)
  const w = 60  // 宽度
  const h = 30  // 高度
  const r = 6   // 圆角半径
  
  // 背景矩形
  g.append('path')
    .attr('d', roundedRectPath(d.x, d.y, w, h, r))
    .attr('fill', getSentimentColor(d.sentiment))
    .attr('stroke', d.is_minority ? '#fbbf24' : '#fff')
    .attr('stroke-width', d.is_minority ? 2 : 1)
  
  // 主题文本
  g.append('text')
    .attr('x', d.x)
    .attr('y', d.y)
    .text(d.name)
    .attr('fill', '#fff')
    .attr('font-size', 10)
    .attr('text-anchor', 'middle')
  
  // 信号数量标签
  g.append('text')
    .attr('x', d.x)
    .attr('y', d.y + 12)
    .text(`${d.signal_count}条`)
    .attr('fill', '#fff')
    .attr('font-size', 8)
    .attr('opacity', 0.8)
})
```

**状态**: ⏳ 待实施

## 🎨 视觉设计

### 节点样式

| 层级 | 形状 | 颜色 | 尺寸 |
|------|------|------|------|
| Agent | 六边形 | 根据stance | 11px |
| Signal | 圆点 | 根据evidence_type | 2.5-4.5px |
| Cluster | 圆角矩形 | 根据sentiment | 60x30px ⭐ |
| Factor | 圆形 | 根据category | 6-16px |
| Target | 菱形 | 金色 | 16px |

### 边样式

| 边类型 | 颜色 | 粗细 | 透明度 |
|--------|------|------|--------|
| agent_signal | #888 | 0.3 | 0.2 |
| signal_cluster | #8b5cf6 | 0.5 | 0.3 ⭐ |
| cluster_factor | #8b5cf6 | 1.5 | 0.6 ⭐ |
| factor_factor | #00d4ff | 1.5 | 0.8 |
| factor_target | #fbbf24 | 2 | 0.9 |

### 聚类节点特殊标记

- **少数派聚类**: 金色边框 `stroke: #fbbf24`
- **高质量聚类**: 发光效果 `filter: drop-shadow(0 0 4px #8b5cf6)`
- **情感标识**: 背景颜色
  - positive: 绿色 `#10b981`
  - negative: 红色 `#ef4444`
  - neutral: 灰色 `#6b7280`
